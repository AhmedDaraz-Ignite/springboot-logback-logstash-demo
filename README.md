# springboot-logback-logstash-demo
How does the logback transfer the log to logstash? Try this demo

在运行本示例之前，请参考[docker-elk](https://github.com/Evan1120/docker-elk) 搭建ELK服务，否则产生的日志将不会输出到logstash中。
